## Description

### Application Functionality

**This applicaiton allows the following tasks to be completed**
* View all departments
* View all roles
* View all employees
* Add a department
* Add a role
* Add an employee
* Update an employee's:
  * First Name
  * Last Name
  * Role
  * Manager
* View employees by manager
* View employees by department
* Delete department
* Delete role
* Delete employee
* View utilized budget
